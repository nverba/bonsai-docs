Movie

new Movie

Constructs a Movie instance
Parameters:
Name Type Argument Description
url String <optional>
 The URL location of the movie to load.
callback Function <optional>
 A callback to be called when your movie has loaded (only called if you passed a `url`). The callback will be called with it's first argument signifying an error. So, if the first argument is `null` you can assume the movie was loaded successfully.
