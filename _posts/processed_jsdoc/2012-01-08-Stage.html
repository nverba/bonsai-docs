Stage

A Stage instance is the root of your movie. There should never be a need to instantiate a new Stage from within a movie.

new Stage

Constructs the movie root, i.e. the global stage.

loadSubMovie

Loads a sub movie and runs it in the context of a new Movie object (should be provided by RunnerContext bootstrap)

Parameters:
Name Type Description
movieUrl

URL of sub-movie (will use the dirname of the submovie as asset path)
callback Function

A callback to be called when your movie has loaded. The callback will be called with it's first argument signifying an error. So, if the first argument is null you can assume the movie was loaded successfully.
