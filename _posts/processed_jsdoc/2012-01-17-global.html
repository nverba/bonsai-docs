request(aRequest) → {AssetDisplayObject|AssetRequest}

Provides support to perform the loading of a Video or Bitmap via HTTP Request and returns the current Video or Bitmap instance.
This:
  • {AssetDisplayObject}
Parameters:
Name Type Description
aRequest String | Array The request needs to accomplish the requirements of AssetRequest
  • runner/asset_display_object.js, line 35
Returns:
The current AssetDisplayObject Istance or an AssetRequest instance
Type
AssetDisplayObject | AssetRequest
Example
myVideo.request();
myVideo.request('../myMovie.mp4');
myVideo.request('http://www.movieCenter.com/myMovie.mp4');
myShape.request([
  {src:'../myMovie.mp4', type:'video/mp4'},
  {src:'../myMovie.ogv', type:'video/ogg'}
]);
myShape.request({
  resources: [
    {src:'../myMovie.mp4', type:'video/mp4'},
    {src:'../myMovie.ogv', type:'video/ogg'}
  ],
  loadLevel: 'can-play'
});
or in case of an image
myBitmap.request();
myBitmap.request('../myImage.jpg');{% endhighlight %}
