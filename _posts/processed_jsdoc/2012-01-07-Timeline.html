Timeline

<static> frames

Adds frames to the timeline. Each frame is a function. Existing frames are overwritten.

The newFrames parameter can be an array where each index represents a frame, or an object, where keys may be a number plus a unit seconds, percents), e.g. 25, 5s, 30%. Percentages relate to the current length of the timeline.
