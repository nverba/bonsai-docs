path. Star

new Star(x, y, radius, rays, factor)

Creates a star
Parameters:
Name Type Argument Description
x Number The horizontal offset/translation of the star center.
y Number The vertical offset/translation of the star center.
radius Number The radius of the star
rays Number The number of rays of the star. Must be > 3
factor Number <optional>
 determines the star "pointiness". 0: all rays start at (0, 0) 1: the star looks like a regular polygon: 3 vertices are on a line. If omitted, a regular star is created
