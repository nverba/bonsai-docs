<static> advancedLinear

Creates a linear gradient

Parameters:
Name Type Argument Description
stops Array | Object

Color stops in the form: ['red','yellow',...] or [['red', 0], ['green', 50], ['#FFF', 100]] i.e. Sub-array [0] is color and [1] is percentage As an object: { 0: 'yellow', 50: 'red', 100: 'green' }
direction Number | String

Direction in degrees or a string, one of: top, left, right, bottom, top left, top right, bottom left, bottom right
matrix Matrix <optional>

Matrix transform for gradient
repeat String <optional>

How many times to repeat the gradient
units String <optional>

Either 'userSpace' or 'boundingBox'.
Returns:

A LinearGradient instance
