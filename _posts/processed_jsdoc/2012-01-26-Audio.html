new Audio(aRequest, callback)

The Audio constructor
Parameters:
Name Type Argument Description
aRequest String | Array The request needs to accomplish the requirements of AssetRequest
callback Function <optional>
 A callback to be called when your movie has loaded (only called if you passed a `aRequest`). The callback will be called with it's first argument signifying an error. So, if the first argument is `null` you can assume the movie was loaded successfully.
